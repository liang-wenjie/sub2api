# Image Generation Compact Select Width Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the model, size, and output-count selects follow their currently selected text width without changing other controls.

**Architecture:** Each target label becomes an inline-grid sizing container containing an inaccessible text mirror and the existing native select in the same grid cell. Vue render data updates the mirror synchronously, while local CSS preserves the native control's padding, dropdown indicator, and responsive maximum width.

**Tech Stack:** Vue 3, TypeScript, CSS, Vitest, Vue Test Utils, Playwright Core
